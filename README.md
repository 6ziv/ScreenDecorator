#ScreenDecorator
一个给屏幕“贴膜”的小型程序。

初步实现的功能:
- 前端显示
- 键鼠透过
- 可以注册为windows屏幕保护。
- 支持自定义图片

  
构建环境:  
*Qt5.11.0  
*Qt Creator  4.6.2（Community)
*MinGW 5.3.0
